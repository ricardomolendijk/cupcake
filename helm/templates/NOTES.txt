GETTING STARTED:

1. Create your first upgrade:

   $ kubectl apply -f - <<EOF
   apiVersion: cupcake.ricardomolendijk.com/v1
   kind: DirectUpdate
   metadata:
     name: upgrade-to-1-27-4
   spec:
     targetVersion: "1.27.4"
     components: [kubeadm, kubelet, kubectl]
     strategy: rolling
     concurrency: 1
   EOF

2. Monitor the upgrade:

   $ kubectl get directupdate upgrade-to-1-27-4
   $ kubectl get directupdate upgrade-to-1-27-4 -o yaml

3. Watch operator logs:

   $ kubectl logs -n {{ .Release.Namespace }} deployment/{{ include "cupcake.fullname" . }} -f

4. Check agent status:

   $ kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/component=agent
